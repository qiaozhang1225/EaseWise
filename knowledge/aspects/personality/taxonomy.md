# Personality Taxonomy

## Allowed levels

- `性格底色很稳`
- `性格整体顺手`
- `有特点，但更吃表达方式`
- `情绪张力偏明显`
- `长期容易拧巴内耗`

## Allowed types

- `主动外放型`
- `务实结果型`
- `细腻感受型`
- `谨慎防御型`
- `敏感拉扯型`
- `强压硬撑型`
- `慢热压抑型`
- `表达存在型`

## Taxonomy rule

- level / type 由代码锁定
- 大模型只负责解释，不负责改 taxonomy
- mixed signals 只能解释主矛盾，不能替换档位
