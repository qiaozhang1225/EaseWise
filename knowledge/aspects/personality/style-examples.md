# Personality Style Examples

These examples are style references, not fixed templates.

## Example 1｜气质底色开法

这个号放在性格上，第一眼给人的不是锋利感，而是一种有自己节奏的底色。它不一定很外放，但会让人感觉这个人有分寸、有边界，不是轻易就被带着走的类型。

## Example 2｜表达落点开法

这个号真正要看的，不只是“会不会说”，而是说出来的时候，是让人觉得舒服，还是让人觉得有压迫感。它的个性不一定弱，问题更多出在表达方式上，用得顺是特色，用得不顺就容易变成棱角。

## Example 3｜情绪承接开法

有些号的问题在表面，有些号的问题在后劲。这个号更像后者：当下不一定立刻爆，但压力一长、关系一熟、事情一多，情绪和感受更容易沉下去，最后变成一种别人不一定看得懂、自己却很累的内耗。

## Example 4｜边界感开法

这个号不是没温度，而是边界感比较重。它更像那种先观察、先判断、先看值不值得的人，不会一上来就把自己全摊开。

## Example 5｜结构点破型开法

这个号真正的问题，不在于它有没有个性，而在于它的个性容易用一种不省力的方式出来。看这种号，最重要的不是判断“强不强”，而是判断“顺不顺”。

## Example 6｜高分但非满格开法

性格分高，不代表一点问题都没有，只代表这股气质的大框架是顺的。这个号之所以还能长期用，是因为主轴没塌；但它没有到满格，说明边角位置还是有扣分。真正要讲清楚的，不是“它性格不错”，而是“它为什么还能顺住，以及还差在哪一口气”。

## Example 7｜客户建议开法

如果你现在最在意的是对外气质、表达舒服度和长期相处感，那这个号就不能只看“有没个性”，而要看“在哪些场景下更容易失衡”。有些号码平时没事，一到高压沟通、关系拉扯或长期相处，问题就会放大。这个号更值得看的，就是它的扣分会不会刚好落在你最在意的性格环节上。

## Example 8｜长期使用感开法

看性格，不是只看亮不亮眼，而是看能不能长期用得省力。前面有表现，不代表后面不累；表面有个性，不代表里面不耗；一时能撑，不代表长期适合硬扛。真正要点破的，往往不是“它像什么样的人”，而是“它会不会让你越用越顺”。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：
- “不是不能用，而是……”
- “问题不在……而在……”
- “重点不在表面……而在……”
- “如果你最近正处在性格敏感期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只能在真的贴切时使用。
