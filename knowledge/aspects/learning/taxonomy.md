# Learning Taxonomy

## Allowed levels

- `学习承接感很稳`
- `学习整体可守`
- `学得进，但更吃节奏和方法`
- `学习波动偏明显`
- `当前不宜把它当学习主用`

## Allowed types

- `系统吸收型`
- `方法整合型`
- `慢热沉淀型`
- `输出驱动型`
- `专注敏感型`
- `压力回拉型`
- `反复卡顿型`
- `前强后沉型`

## Taxonomy rule

- level / type 由代码层锁定
- 大模型只负责解释，不负责改 taxonomy
- 如果 evidence 互相重叠，只解释主矛盾，不改档位
