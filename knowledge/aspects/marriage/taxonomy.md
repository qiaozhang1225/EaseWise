# Marriage Taxonomy

## Allowed levels

- `婚姻承接感很稳`
- `婚姻整体可守`
- `能走下去，但更吃相处方式`
- `关系扰动偏明显`
- `当前不宜在婚姻上硬扛`

## Allowed types

- `稳定陪伴型`
- `慢热磨合型`
- `沟通拉扯型`
- `桃花扰动型`
- `权责失衡型`
- `外界牵动型`
- `高压消耗型`
- `前合后松型`

## Taxonomy rule

- level / type 由代码层锁定
- 大模型只负责解释，不负责改 taxonomy
- 如果 evidence 互相重叠，只解释主矛盾，不改档位
