# Marriage Style Examples

These examples are style references, not fixed templates.

## Example 1｜长期陪伴感开法

这个号码放在婚姻里，重点不只是“有没有感情”，而是“这段关系能不能长期接得住”。有些号前面容易热，后面不一定稳；有些号不是一开始就很黏，但越相处越有陪伴感。真正值得看的，是它有没有把长期过日子的那口气接起来。如果这口气在，关系就更容易守得住。

## Example 2｜高分但非满格开法

婚姻分高，不代表一点问题都没有，只代表这段关系的大框架还站得住。这个号之所以还能长期用，是因为婚姻主轴没塌；但它没有到满格，说明边角位置还是有扣分。真正要说清楚的，不是“它婚姻不错”，而是“它为什么还能守住，以及还差在哪一口气”。

## Example 3｜边界信任开法

这类号码最怕的，不一定是没感情，而是边界感和安全感不够稳。平时看着还好，一旦外界人事、异性往来或者关系敏感点被碰到，原本不算大的问题就容易被放大。这样的盘，不是完全不能走，而是要比别人更重视边界和信任的经营。

## Example 4｜权责失衡开法

这个号在婚姻里更怕的，不是没有机会，而是两个人一旦进入真正过日子的阶段，谁来扛、谁来让、谁来主导，容易慢慢变成关系里的隐性拉扯。前面靠感觉还能接，后面涉及分工、责任和现实安排时，就更容易显出扣分。

## Example 5｜前合后松开法

有些号码前面很容易接上，甚至一开始会让人觉得有默契、有黏性，但真正进入长期相处后，后段能不能稳住，反而是更大的问题。这个号更像是前面能合，后面容易松，所以扣分不一定先出在开始，而是出在长期承接。

## Example 6｜高压消耗开法

这不是那种一下子就断掉的号码，它更像是慢慢耗。刚开始也许还能顶，甚至外人看不出太大问题，但关系一旦进入高压阶段，翻旧账、情绪反复、心累和长期消耗就会一点点积起来。这样的结构，最怕的不是短期波动，而是长期一直硬扛。

## Example 7｜客户建议开法

如果你现在最在意的是婚姻稳定、伴侣配合和长期家庭节奏，那这个号就不能只看“能不能用”，而要看“在哪些关系场景下更容易失衡”。有些号码平时没事，一到谈婚论嫁、婚后分工或情绪敏感期，就容易放大问题。这个号更值得看的，就是它的扣分会不会刚好落在你最在意的婚姻环节上。

## Example 8｜结构点破型开法

看婚姻，不是只看有没有缘分，而是看能不能把缘分守成关系，把关系守成日子。前面能热，不代表后面不散；表面有黏性，不代表里面没有拉扯；一时能接，不代表长期能守。真正要点破的，往往不是一开始那点感觉，而是这段关系在时间里会不会越走越省心。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：
- “不是不能用，而是……”
- “问题不在……而在……”
- “重点不在感觉……而在承接……”
- “如果你最近正处在婚姻敏感期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只能在真的贴切时使用。
