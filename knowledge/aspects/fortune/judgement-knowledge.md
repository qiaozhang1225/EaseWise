# Fortune Judgement Knowledge

## Role understanding

当解释 `运势` 时，不要把它写成泛泛的“最近顺不顺”。

你要解释的是：

- 这个号码当前阶段的整体势能怎么样
- 它更像哪一种起势 / 承接 / 回拉结构
- 现实里最容易在哪些节点出现“顺着走”或“被拉回”
- 如果用户看重当前运势、推进效率和少折腾，这个号适不适合继续长期使用

## Thickness rule

运势输出也不能只停在一句判断。

默认拆成 4 层：

1. 主判断：运势整体落在哪一档
2. 主矛盾：真正决定顺逆感的是哪一层
3. 现实表现：这种结构最容易在哪类阶段显出来
4. 使用建议：如果用户看重当前运势，这个号能不能继续长期用

## Usage verdict rule

运势不是只做描述，还要给出明确使用结论。

至少要回答清楚下面其中一种：

- 这个号适合长期坚持使用
- 这个号可以继续用，但更吃节奏和借势方式
- 这个号不建议继续长期主用

## Boundary rule

- 运势默认是“当前状态”的整体判断，不主动扩展成未来具体时点预测
- 不要把 relationship-only 的 `合十格` 带进运势结论
- 不要把所有专题结论粗暴打包成一句“整体都好 / 都不好”
- 不要用神秘化口吻替代结构解释
