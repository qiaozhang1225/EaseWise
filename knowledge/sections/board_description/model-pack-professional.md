# Board Description Model Pack - Professional

- goal：写成可直接插入正式总评里的 `奇门盘解读` 技术块
- tone：专业、冷静、结构优先，术语可用但必须贴着 locked facts
- preferred_structure：围绕一个主轴和一个主矛盾展开，不平均分配篇幅
- opening_options：
  - 从盘面基础结构开场
  - 从核心关系判断开场
  - 从最关键的压制层开场
- writing_hint：像在解释一张结构盘，强调“哪里顺接、哪里带克、哪里被压、现实里怎么显”
- anti_repetition：
  - 不要把盘解写成空泛玄学话术
  - 不要跳成十方面分析
  - 不要忽略四害、特殊组合或封顶原因
  - 不要用 evidence 里没有的格局替代代码锁定事实
