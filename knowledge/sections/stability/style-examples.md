# Stability Style Examples

These examples are style references, not fixed templates.

## Example 1

适合长期使用。这个号的底层承接感是能站住的，虽然不是每个细节都满格，但主要结构没有明显把长期状态拉垮的硬伤。作为常用号继续用问题不大，真正要注意的是关键阶段别过度冒进，把它放在稳定推进、持续积累的节奏里更容易发挥出来。

## Example 2

可以继续使用，但要注意使用方式。这个号不是不能用，它的问题在于某些扣分项会在长期里慢慢显出来，尤其到了需要收口、沉淀或维持关系的时候，会比短期使用更容易感到反复。如果只是日常联系或普通业务承接，可以继续用；但如果你特别看重被扣分的方向，就不要把它当成完全稳定的主号。

## Example 3

不建议继续长期主用。这个号短期未必马上出大问题，但长期放在主号位置上，容易把盘面里的反复、消耗或关系牵动放大。尤其当你把它用于事业推进、财务合作或重要关系维护时，问题会更明显，所以更适合作为辅助使用，而不是继续承担主要号码角色。

## Example 4

不建议长期使用，请尽快调整。这个号的长期承接力偏弱，关键扣分项不是边角小问题，而是会持续影响状态、节奏和结果保有。继续硬用容易越用越耗，尤其在你特别在意的方向上更容易反复出问题，所以建议尽快调整，不要把它继续作为长期主用号码。

## Anti-pattern Reminder

不要写成：
- 空亡如何，门迫如何，入墓如何，击刑如何
- 这个号码属于某个内部评分档位
- 根据课程或知识库判断
- 综合来看较为稳定但仍需注意
- 如有需求可进一步咨询
