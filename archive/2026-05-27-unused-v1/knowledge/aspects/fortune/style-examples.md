# Fortune Style Examples

These examples are style references, not fixed templates.

## Example 1｜起势开法
这个号码看运势，重点不只是“眼前有没有一点起色”，而是这股起色能不能顺着接下去。有些号前面气口一开，后面承接也跟得上；有些号表面看着有势，真正进到关键节点，反而更容易被现实问题往回拉。真正要看的是，它的顺势感能不能落地，而不是表面热不热闹。

## Example 2｜有势但不满格
运势分不低，不代表一路都顺，只代表这股势的大框架还站得住。这个号之所以还可以继续用，是因为主轴没塌；但它没到满格，说明最容易拉后腿的那一层还在。真正要讲清楚的，不是“它有势”，而是“它为什么还能走，以及它到底卡在什么位置”。

## Example 3｜风险回拉型
这个号更怕的不是一点机会都没有，而是机会一来，动作一放大，风险回拉就跟着出来。前面能起，后面容易抖；表面能推，临门容易卡。它不是彻底没势，而是势头需要有人去接、去稳、去修正，不能拿它当完全顺风的盘去硬冲。

## Example 4｜前通后滞型
有些号码不是前面起不来，而是越往后越发沉。你会发现它开局不差，甚至一开始挺有感觉，但到了持续推进、长期扛事、收尾落地这些环节，速度和承接感就明显往下掉。这类盘最值得看的不是“会不会起”，而是“起了以后能撑多久”。

## Example 5｜外部牵动型
这个号的运势不完全是自己单线往前推出来的，它更容易被关系、人情、合作对象和外部节奏带着走。环境一顺，它会显得很好用；环境一乱，原本还能接住的势也容易被拖偏。所以它的重点不是单看自己够不够强，而是看外部变量一进来时，节奏会不会被带乱。

## Example 6｜建议开法
如果你现在最在意的是少折腾、推进顺、关键节点别掉链子，那这个号就不能只看表面有没有起色，而要看它的扣分是不是正好落在你最在意的阶段上。有些号平时没事，一到放大动作、谈结果、冲节点，问题就开始显出来。这个号值不值得继续用，关键就看这层代价你能不能接受。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：

- “不是没有势，而是……”
- “重点不在表面，而在后段……”
- “如果你最近正处在敏感期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只在真的贴切时使用。
