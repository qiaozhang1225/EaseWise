# Suitable Job Style Examples

These examples are style references, not fixed templates.

## Example 1｜岗位匹配开法

这个号码看适合职业，重点不只是“能不能干活”，而是“放到什么岗位、什么分工、什么平台里更容易把能力稳定发挥出来”。有些号能力不差，但岗位一不对路就会越做越耗；有些号不是所有角色都能打满，但一放到对的位置，整个人就会顺很多。真正值得看的，是它的工作方式、角色分工和环境承接能不能一起站住。

## Example 2｜还能用但不满格

职业适配分不低，不代表什么岗位都能顺，只代表这套职业主轴还没塌。这个号之所以还能继续用，是因为它不是没有职业优势；但它没到满格，说明最容易拉后腿的平台、边界或长期落点层还在。真正要讲清楚的，不是“它还行”，而是“它为什么还能顺着做，以及它到底差在哪”。

## Example 3｜管理统筹型

这类号码不是只会埋头做事，它更适合把方向、节奏和协同一起抓起来。前面可能不是靠嘴热闹，而是靠结构感、判断力和控盘感把事情慢慢推顺。它真正怕的，不是事情多，而是岗位边界乱、平台承接差，导致原本能带起来的优势被环境消耗掉。

## Example 4｜风控排障型

有些号码不是最适合走轻松前台路线，而是更适合放在看风险、查问题、处理异常、快速收口的位置上。它的价值不在“场面多好看”，而在事情一出状况时能不能先看到问题、再稳住局面。这样的盘最怕的，不是压力本身，而是长期高压下没有足够支持，最后把职业优势拖成职业消耗。

## Example 5｜流程审核型

这类号最怕的，不一定是岗位普通，而是角色过于混乱。规则清晰、流程明确、职责边界稳的时候，它反而很能守住质量和闭环；可一旦岗位要求它既前台外放、又后台细抠，还要一路顶着跑，它的代价就会开始冒出来。真正要看的是，它适不适合这个工作方式，而不是表面头衔好不好听。

## Example 6｜建议开法

如果你现在最在意的是职业匹配、长期定位和少走弯路，那这个号就不能只看“能不能做”，而要看它的扣分是不是正好落在你最在意的岗位环节上。有些号平时没事，一到换平台、升角色、扛压力、拉长周期，问题就开始显出来。这个号值不值得继续用，关键就看这层代价你能不能接受。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：

- “不是不能做，而是……”
- “重点不在能力，而在环境……”
- “如果你最近正在考虑换工作……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只在真的贴切时使用。
