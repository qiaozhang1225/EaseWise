# Learning Style Examples

These examples are style references, not fixed templates.

## Example 1｜学习承接开法

这个号码看学习，重点不只是“前面能不能听懂”，而是“学进去的东西它接不接得住，过两天还能不能提出来”。有些号理解不慢，但留存不稳；有些号不是一开始最猛，却越学越扎实。真正值得看的，是它的学习承接、节奏和复盘感能不能一起站住。

## Example 2｜还能用但不满格

学习分不低，不代表一点波动都没有，只代表这套学习主轴还没塌。这个号之所以还能继续用，是因为它不是完全学不进去；但它没到满格，说明最容易拉后腿的专注、方法或复盘层还在。真正要讲清楚的，不是“它还行”，而是“它为什么还能学得住，以及它到底差在哪”。

## Example 3｜方法整合型

这类号码不是靠硬背取胜，而是更擅长把零散的东西梳顺、归类、搭出框架。前面一旦找到学习路径，后面吸收会越来越顺。它真正怕的，不是内容太难，而是顺序乱了、方法没抓住，导致本来能学明白的东西也被学散了。

## Example 4｜专注敏感型

有些号码不是没学习力，而是太容易被外界节奏带乱。前面一安静、一专注，它是能进去的；但一旦任务切碎、信息太多、情绪上来或者环境反复打断，效率就会明显上下波动。这样的人学习真正吃的，不是智力差距，而是专注节奏有没有被守住。

## Example 5｜前强后沉型

这类号最怕的，不一定是起步慢，而是后面越来越沉。你会发现它一开始也许能跟上、能听懂，甚至还挺有感觉，但一到长期复盘、反复提取、后段巩固这些环节，速度和留存就开始掉。这类盘最值得看的不是“会不会学”，而是“学了以后能不能留下来”。

## Example 6｜建议开法

如果你现在最在意的是学习效率、专注稳定、复盘留存和少反复，那这个号就不能只看“有没有一阵子的学习感觉”，而要看它的扣分是不是正好落在你最在意的学习环节上。有些号平时没事，一到长期输出、复盘或高压阶段，问题就开始显出来。这个号值不值得继续用，关键就看这层代价你能不能接受。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：

- “不是学不会，而是……”
- “重点不在表面，而在后段……”
- “如果你最近正处在备考期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只在真的贴切时使用。
