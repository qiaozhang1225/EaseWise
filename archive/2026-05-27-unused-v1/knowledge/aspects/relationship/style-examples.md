# Relationship Style Examples

These examples are style references, not fixed templates.

## Example 1｜关系承接开法

这个号码看人际感情，重点不只是“有没有人缘”，而是“来的人它接不接得住，熟了以后会不会越处越硬”。有些号前面热闹，后面不一定舒服；有些号不是一开始特别会来事，但熟了以后反而更稳。真正值得看的，是它的关系承接和边界感能不能一起站住。

## Example 2｜还能用但不满格

人际感情分不低，不代表一点摩擦都没有，只代表这套关系主轴还没塌。这个号之所以还能继续用，是因为它不是完全不接人；但它没到满格，说明最容易拉后腿的边界、回应或情绪层还在。真正要说清楚的，不是“它还行”，而是“它为什么还能接得住，以及它到底差在哪”。

## Example 3｜边界敏感型

这类号码最怕的，不一定是没人来往，而是关系一热、距离一近、边界一模糊，本来不大的问题就容易被放大。平时看着还能处，一到暧昧、人情、异性往来或长期互动阶段，分寸感就会决定这段关系是越处越顺，还是越处越硬。

## Example 4｜强势压场型

有些号码不是没有人缘，而是表达和主导感太强。前面可能会让人觉得有存在感、能推动、讲结果，但相处久了，对方会慢慢感觉被压着、被带节奏，或者不太容易轻松做自己。这样的人际结构，不一定缺连接，缺的是柔和感和舒服度。

## Example 5｜情绪反复型

这类号不一定是关系少，而是关系里更容易出现敏感、多想、误解和一下热一下冷。平时靠直觉和气场还能推进，一到真正需要稳定回应、稳稳接住彼此的时候，情绪回拉就会变成隐性成本。

## Example 6｜外热内耗型

有些号码表面上并不冷，甚至还挺热闹，问题在于越往后越累。前面有来有往不代表后面省心，热闹感也不等于舒服感。这样的盘最怕的，不是短期没有互动，而是长期总在修、总在磨、总在反复解释。

## Example 7｜建议开法

如果你现在最在意的是关系舒服度、人缘稳定、边界清楚和少内耗，那这个号就不能只看“有没有人理你”，而要看它的扣分是不是正好落在你最在意的相处环节上。有些号平时没事，一到长期互动、暧昧升温或关系修复阶段，问题就开始显出来。这个号值不值得继续用，关键就看这层代价你能不能接受。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：

- “不是没人缘，而是……”
- “重点不在表面，而在后段……”
- “如果你最近正处在感情敏感期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只在真的贴切时使用。
