# Wealth Style Examples

These examples are style references, not fixed templates.

## Example 1｜进财承接开法

这个号码放在财运里，重点不只是“有没有来钱”，而是“来的钱能不能接得住、留得下”。有些号机会不少，但后段留不住；有些号不是一下子爆发，却能慢慢把财路做稳。真正值得看的，是它有没有把进财和留财接成一条线。

## Example 2｜高分但非满格开法

财运分高，不代表一点问题都没有，只代表这条财路的大框架还站得住。这个号之所以还能长期用，是因为主轴没塌；但它没有到满格，说明边角位置还是有扣分。真正要说清楚的，不是“它财运不错”，而是“它为什么还能做，以及还差在哪一口气”。

## Example 3｜资源整合型开法

这类号码不太像单纯守小财的结构，它更像适合靠平台、资源、项目、统筹和结果导向去把财路放大。问题不是能不能赚，而是扩张动作如果快过了承接能力，财运就容易从“能做大”变成“波动大”。所以它要看资源和节奏，不是只看胆子。

## Example 4｜合作分利型开法

这个号在财运里更怕的，不是没有项目，而是有项目以后，合作怎么分、责任怎么落、利益怎么切，容易慢慢变成真正的扣分点。前面看着是机会，后面如果分利结构不清，钱未必真能留到自己手里。

## Example 5｜现金流波动型开法

有些号码不是没财，而是钱走得不顺。机会能来，项目能起，但回款、周转、节奏和中间消耗更容易忽上忽下。这样的盘，问题通常不在“有没有财路”，而在“财路能不能顺着落到口袋里”。

## Example 6｜前强后漏开法

这个号更像前面能见到钱、能看到机会，但后段守财、留财和持续收益的能力不是满格。很多人会误以为“能赚到”就等于“适合长期用”，但真正要看的是，赚来的结果最后能不能沉下来、留得住。

## Example 7｜客户建议开法

如果你现在最在意的是现金流稳定、合作回款和长期收益，那这个号就不能只看“能不能赚钱”，而要看“在哪些财务场景下更容易失衡”。有些号码平时没事，一到扩项目、做分账、追回款，问题就会放大。这个号更值得看的，就是它的扣分是不是刚好落在你最在意的财务环节上。

## Example 8｜结构点破型开法

看财运，不是只看有没有财门，而是看财门开了以后，后面能不能承、能不能守、能不能留。前面能进，不代表后面不漏；表面有势，不代表里面没耗；一时能冲，不代表长期适合硬扛。真正要点破的，往往不是“有没有财”，而是“这条财路值不值得长期压上去”。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：
- “不是不能赚钱，而是……”
- “问题不在进财，而在留财……”
- “重点不在表面机会，而在后段承接……”
- “如果你最近正处在财务敏感期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只能在真的贴切时使用。
