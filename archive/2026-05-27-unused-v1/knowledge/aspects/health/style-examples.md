# Health Style Examples

These examples are style references, not fixed templates.

## Example 1｜承压开法
这个号码看健康，重点不只是眼前扛不扛得住，而是这份承压感能不能持续、恢复能不能跟上。有些号平时看着没事，一到连轴转、长期操心、作息乱掉的时候，问题就会开始显出来。真正要看的是，它的精力承接能不能落地，而不是表面上有没有在撑。

## Example 2｜还能用但不满格
健康层不算差，不代表一点消耗都没有，只代表这套承压结构还没塌。这个号之所以还可以继续用，是因为主轴没断；但它没到满格，说明最容易拉后腿的节律或恢复层还在。真正要讲清楚的，不是“它还行”，而是“它为什么还能扛，以及它到底差在哪”。

## Example 3｜睡眠波动型
这类号不一定是一下子爆问题，而是更容易在睡眠、紧绷、精神松不开和恢复不透这几层上反复。平时靠意志还能顶，越到高压、连续推进和长期熬着扛的时候，状态越容易一点一点往下掉。所以重点不是单看“有没有病”，而是看“这份消耗会不会越用越透”。

## Example 4｜恢复偏慢型
有些号码不是前面扛不住，而是后面恢复得慢。你会发现它出力时未必差，甚至短期还能顶一阵，但一旦进入连续消耗、反复劳心或高压收尾这些阶段，恢复感就明显跟不上。这类盘最值得看的不是“能不能扛”，而是“扛完以后回不回得来”。

## Example 5｜情绪牵动型
这个号的健康承载不完全是身体单线的问题，它更容易被情绪、人际和外部节奏带着走。外面一乱，里面就更难松；关系一牵动，睡眠、节律和恢复也容易跟着掉。所以它的重点不是单看自己够不够强，而是看外部变量一进来时，状态会不会被带乱。

## Example 6｜建议开法
如果你现在最在意的是少透支、睡得稳、恢复快，那这个号就不能只看表面能不能扛，而要看它的扣分是不是正好落在你最在意的节律环节上。有些号平时没事，一到高压、熬夜、连续推进，问题就开始显出来。这个号值不值得继续用，关键就看这层代价你能不能接受。

## Anti-pattern reminder

不要让所有输出都长成同一种样子。

尤其不要每次都重复：

- “不是生病，而是……”
- “重点不在表面，而在后段……”
- “如果你最近正处在敏感期……”
- “空亡、门迫、入墓、击刑分别如何如何……”

这些句式可以用，但只在真的贴切时使用。
