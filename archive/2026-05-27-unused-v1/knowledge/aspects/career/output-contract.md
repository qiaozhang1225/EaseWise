# Career Output Contract

## Required fields

When this aspect is rendered as structured output, keep these fields:

- `career_level`
- `career_type`
- `core_judgement`
- `real_world_manifestation`
- `advice`
- `user_facing_paragraph`

## Paragraph rule

最终展示段落应像人说的话，不能像：
- 分数 dump
- taxonomy 标签复读
- 四害 checklist

## Minimum quality bar

用户读完以后，至少应该知道：
- 这个号在事业上能不能长期用
- 主要扣分落在哪一层
- 现实里会怎么表现
- 哪些场景更值得留意
- 应该怎么用，而不是只知道一个标签
