# Personality Judgement Knowledge

性格专项先看格局主追求，再看日支主气十神的脾气、天干十神的外显印象、地支十神的内在真实、缺失十神的人格短板。

判断要点：
- 正财、偏财偏资源与财富追求；食神、伤官偏表达、享受和才华追求；正印、偏印偏学习、文化、信念和特殊洞察；正官、七杀偏秩序、身份、挑战和执行。
- 日坐七杀反应强、面子敏感；日坐食伤急来急去、爱憎分明；日坐比劫倔强、不服输；日坐官印较温和守规；日坐财才主观自信；日坐偏印慢热疏离。
- 十神缺失只写倾向，不写人格缺陷判决；十神过多要结合喜忌说明“可用处”和“过度处”。
- 强弱代表自信心与承载力，不等同于好坏。身强重行动与掌控，身弱重环境承接与支持系统。

## Personality Explicit Rule Pack

性格专项必须从“日主承载力 + 日支主气 + 主导十神 + 喜忌”合看，不只按某一个十神贴标签。日主偏弱时，现实里常表现为对环境、平台、节奏和他人反馈更敏感，机会太多反而容易累；日主偏强时，现实里更有主张、行动和掌控欲，但要提醒合作边界和听取反馈。日支是用户更私下、更本能的反应方式，天干十神是外界第一眼看到的表达，地支藏干是内在真实动机。

十神要翻译成现实行为：印星多，写学习、保护、证书、长辈资源和安全感；食伤多，写表达、创意、作品、享受和对规则的挑战；财星多，写资源意识、现实感、效率和经营心；官杀多，写责任、规则、压力、竞争和目标感；比劫多，写自我、同辈、朋友、竞争和不服输。缺某类十神时，只写该类能力需要后天训练，例如缺印不是没学识，而是需要稳定学习系统；缺财不是没钱，而是资源配置意识要靠现实训练。

特殊结构只作性格倾向：伤官见官代表表达和规则容易摩擦，现实里要注意说话方式、上级关系和制度边界；枭神夺食代表想得多、输出被打断或快乐感下降，现实里要用规律作品、运动和社交保持流动；官杀混杂代表稳定规则与高压竞争并存，现实里容易既想安全又被目标推着走。结论要落到沟通、决策、抗压、学习、合作和情绪管理。
