# Marriage Judgement Knowledge

婚姻专项看配偶星、日支夫妻宫、官杀/财星状态、比劫争合、食伤制官、印星承接，以及合冲刑害对亲密关系节奏的影响。

判断要点：
- 男命以财星为重要关系资源，女命以官杀为重要关系资源；不要绝对化，应结合日主强弱、喜忌和夫妻宫。
- 夫妻宫稳定且为喜，现实中更容易形成清楚边界、长期合作和互相支持；夫妻宫受冲刑害，更多体现为关系节奏反复、沟通成本高或家庭事务牵动。
- 官杀太重或财星太重而日主承载不足，容易在关系里感到压力、被安排或为了关系承担过多现实责任。
- 食伤旺时要解释为表达欲、主见和对关系质量的要求，不能简单写“克夫/克妻”；若能被印星承接，反而适合通过沟通和共同成长改善关系。
- 比劫明显时可写同辈竞争、圈层干扰、消费观或边界感带来的关系消耗，不写第三者确定事件。

## Marriage Explicit Rule Pack

婚姻专项必须按“夫妻宫 + 配偶星 + 日主承载 + 喜忌 + 合冲刑害 + 岁运触发”合断。夫妻宫是日支，代表共同生活的位置、相处节奏和关系稳定度；男命财星是重要关系资源，女命官杀是重要关系资源，但不能脱离喜忌和强弱。夫妻宫为喜且少刑冲，现实里更容易互相支持、边界清楚、长期合作；夫妻宫为忌或受冲刑害破，更多表现为沟通反复、距离变化、家庭事务牵动、现实责任压力，不直接写离婚定局。

关系压力标签要转译：男命比劫多，写同辈竞争、朋友牵扯、金钱边界或自我主张影响关系；女命食伤多，写表达锋芒、要求高、对关系质量敏感，不写克夫。财才混杂或官杀混杂，写外缘、机会、标准不稳定和关系选择牵动；桃花只代表吸引力、审美和关系机会，不单点判断出轨。夫妻宫暗合可以写隐性牵连或沟通不透明，但必须保持概率表达。

时间判断只写风险窗口：夫妻宫低分且岁运刑冲破害夫妻宫，或男命岁运见比劫、女命岁运见食伤，可提示关系经营压力上升；日元逢合、夫妻宫逢合、桃花年或冲被制住，可提示关系推进机会。输出要落到沟通方式、共同财务、边界、家庭介入和重大关系选择。
