# Health Judgement Knowledge

健康专项看五行偏枯、七杀、偏印、刑冲、被冲伤五行和宫位身体。

判断要点：
- 七杀为实际病痛和压力源；偏印为经络淤堵、隐性不舒和慢性卡点。
- 某五行三个及以上，或某五行被冲伤，该五行及其所克五行作为健康风险提示。
- 天干四冲、地支六冲中被冲伤五行要重点说明。
- 宫位身体可用于定位：年柱偏头颈早年，月柱偏胸腹青年，日柱偏核心身体中年，时柱偏腿脚晚年。
- 情绪与脏腑：思伤脾、恐伤肾、怒伤肝、喜伤心、忧伤肺。
- 七杀能量被三刑、自刑、合局、会局、冲局或岁运突然加强时，健康压力上升；只写风险管理和保养建议。
- 灾煞、天罗地网等高风险神煞只作为安全提醒，不得写血光、牢狱、重病等确定断语。
- 所有健康输出只能是保养提醒，不能写医学诊断。

## Health Explicit Rule Pack

健康专项只能写风险倾向和生活管理，不能写诊断、确诊、寿命和必然灾病。判断顺序是：五行偏枯或过旺 -> 被冲伤五行 -> 七杀/偏印位置 -> 宫位身体 -> 岁运触发。某五行三个及以上，或某五行被天干四冲、地支六冲冲伤，该五行及其所克五行作为保养重点。天干身体可作提示：甲肝、乙胆、丙小肠、丁心血、戊胃、己脾、庚大肠筋骨、辛肺、壬膀胱、癸肾；必须转成作息、饮食、压力和运动建议。

七杀代表实际压力源、病痛压力、强制性事件和高压环境；偏印代表隐性不舒、慢性卡点、经络淤堵和想太多。七杀或偏印落在哪个五行、哪个宫位，就提示对应生活系统更容易被消耗。年柱偏头颈早年，月柱偏胸腹青年，日柱偏核心身体和中年，时柱偏腿脚、晚年和长期计划。情绪映射可用：思伤脾、恐伤肾、怒伤肝、喜伤心、忧伤肺。

岁运三刑、自刑、六合、三合、三会、辰戌冲、丑未冲或补齐原局缺字导致七杀、食伤、偏印突然加强时，只写“健康安全和压力管理窗口”。灾煞、元辰、天罗地网、羊刃、飞刃、亡神等只转译为安全、规则、情绪和身体压力，不写血光、重病或意外定局。
