# Pattern Judgement Knowledge

格局专项解释正印、偏印、食神、伤官、正财、偏财、正官、七杀、比肩、劫财等结构如何落到现实表现。重点是把格局翻译成资源获取、表达方式、压力处理和成长路径。

判断要点：
- 正印偏学习、保护、资质、长辈资源和稳定承接；现实中像依靠知识体系、证书、平台或贵人支持成长。
- 偏印偏洞察、非标学习、独立研究和敏感度；现实中适合深度钻研，但要避免过度怀疑和孤立。
- 食神偏稳定输出、技能、作品和温和表达；伤官偏锋利表达、创新、突破规则和强主见，要说明如何把锋芒转成作品或专业能力。
- 正财偏稳定收益、长期经营和责任感；偏财偏机会、交易、资源整合和风险意识。
- 正官偏规则、职位、秩序和责任；七杀偏压力、竞争、执行力和危机处理，必须说明如何用印星/食伤承接，不写成凶断。
- 比肩劫财偏自我、同辈、团队、竞争和资源分配；要解释成合作方式与边界管理。

## Pattern Explicit Rule Pack

格局专项不是报一个名称，而是解释命盘主要力量如何配合，现实里对应资源获取、表达方式、压力处理和成长路径。先看月令和日主强弱，再看主导十神、吉神难神、喜忌比例、制化关系和关键岁运能否接上。身弱盘的核心是扶身、借平台、借贵人、借团队和建立节奏；身强盘的核心是输出、担责、拿资源、受规则约束和把能量流通出去。

结构组合要翻译：伤官配印，是表达、才华、突破和学习资质相互配合，现实里适合把锋芒变成专业权威；伤官见官，是表达和规则相冲，现实里要注意上级、制度、合同、口舌和评价体系；杀印相生，是压力、竞争和平台资质相生，现实里适合高压组织、专业证书和管理路线；食神制杀，是用稳定技能和方法化解压力；枭神夺食，是敏感钻研压住稳定输出，现实里要防想太多、快乐感下降和作品中断。

财官印食比劫都要落到现实路径：财是资源和经营，官杀是规则与压力，印是资质和保护，食伤是作品和表达，比劫是团队与竞争。墓库、空亡、合冲刑害决定格局能否兑现：库打开偏资源释放，墓打开偏承接下降，空亡偏兑现弱。结论要说“怎么用这个格局”，不写格局高低定命。
