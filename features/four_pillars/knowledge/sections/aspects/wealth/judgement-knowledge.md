# Wealth Judgement Knowledge

财富专项看正财、偏财、食伤生财、比劫夺财、日主承载和财星喜忌。

判断要点：
- 正财偏稳定收入、长期经营、工资和合法收益；偏财偏商业机会、投资、信息差和外部资源。
- 食伤生财代表表达、技能、产品、内容或服务带来资源。
- 比劫克财代表朋友、合伙、同辈竞争导致钱财流失。
- 财星为喜且有根、有食伤生助，财富承接更稳；财星为忌或过多，日主承载不足，容易辛苦求财或因财生压。
- 偏财强且岁运再见偏财，可以写机会感增强，但必须提醒风险管理，不能承诺横财。
- 财富时机先看日元能量和财星能量：身强财旺遇食伤财才为强机会，身强财弱为中等机会，身弱财旺遇印枭比劫为强机会，身弱财弱遇印枭比劫为中等机会。
- 四柱无财且身弱，先写求财难度和承载不足；四柱无财但身强，岁运见财可有进账，但不一定长久。
- 比劫库和财才库与个人发财、存钱能力关系最直接；日支或时支带相关库，或岁运遇相关库并被刑冲破害打开，可写资源释放和存钱机会。
- 五鬼为偏财机会与是非风险并见，禄神为职位、贵人、稳定收入和承接力；两者都不能写成确定发财。

## Wealth Explicit Rule Pack

财富专项必须先判断“钱从哪里来、能不能接住、会在哪里漏”。财星代表钱、资源和现实交换，但财多不等于富：身弱财旺，现实里常见机会、责任、客户、伴侣资源或赚钱压力很多，自己却容易被钱事推着走；身强财旺，才更能主动配置资源。食伤生财是靠技能、产品、内容、服务、表达和经营把能力变成收入；官杀生印再生身时，更多是组织、职位、资质和平台带来的稳定承接。

财库和墓库要解释清楚：辰戌丑未既可能是库，也可能是墓。财库被刑冲害破打开，可以写资源释放、存钱机会、资产配置窗口或财务波动；如果是财星入墓或承载不足，要写钱被压住、回款慢、责任重，不写一定发财。比劫夺财要落到合伙分账、人情借贷、朋友消耗、同辈竞争和冲动消费；比劫为喜时也可以写团队协作和朋友助力，但仍要提醒账目清楚。

岁运触发财星、食伤、印枭、比劫、禄神、五鬼时，写成“机会感增强或风险同时上升”。五鬼运财只能表达偏财机会与是非损耗并见，不能承诺横财；禄神为用可写职位、工作、贵人、学业和稳定收入承接增强，禄神空亡或受冲克则兑现力下降。输出要给现金流、合作边界、预算、回款和风险纪律建议。
