# Fengshui Judgement Knowledge

风水专项基于四柱中的五行偏向、喜忌、居住/办公环境象意和家庭环境知识做生活化提醒。不能写恐吓式风水断语，也不能替代现场勘察。

判断要点：
- 五行不足或过旺可以映射到环境调节：木对应植物和成长感，火对应光照和热度，土对应稳定收纳和空间承载，金对应秩序、金属和规则感，水对应流动、通风和信息流。
- 喜用元素可作为环境优化方向，忌神过旺则提醒减少同类环境刺激。
- 合冲刑害可写成居住/办公中容易被噪音、通道、动线、强光、杂乱或频繁变动影响，不写确定灾祸。
- 家宅成员方位知识只作为提醒：例如父母、伴侣、子女和卧室/办公位的关系要回到采光、通风、动线、安静度和收纳秩序。
- 输出要给低成本可执行建议，如整理动线、增强采光、减少杂物、区分休息区和工作区。

## Fengshui Explicit Rule Pack

风水专项只做命盘环境象意和生活空间建议，不替代现场风水实勘。判断顺序是：喜忌五行 -> 五行偏枯 -> 年月地支环境象意 -> 合冲刑害带来的动线压力 -> 家庭成员方位提醒。喜木可写植物、成长感、学习与通风；喜火写采光、热度、曝光和活力；喜土写稳定、收纳、承载和规律；喜金写秩序、工具、规则和整洁；喜水写流动、信息、通风和适度湿润。忌神过旺则提醒减少同类刺激。

地支环境象意用于早年、祖上、住处或坟地周边倾向：子为小溪水井，丑为坟地池塘井庙，寅为树林，卯为花园库房，辰为水库井，巳为阳光热源，午为火堆田园道观，未为田园草地，申为道路机器厂房，酉为工具仓库，戌为庙马路空地，亥为河湖海稻田。必须注明这是地支取象，不是现场结论。

合冲刑害转译为空间压力：冲为动线、噪音、搬动、变化；刑为长期不顺手和重复摩擦；害为隐性不舒服；破为收纳、边界或功能区破损。家宅方位只给低成本建议：采光、通风、动线、收纳、休息区和工作区分离，不能写恐吓式化解、灾祸或绝对风水断语。
