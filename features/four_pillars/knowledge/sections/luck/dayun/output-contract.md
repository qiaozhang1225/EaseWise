# Dayun Output Contract

必须输出 JSON object：
- `cycle_key`: 必须等于 locked facts 中的 `selected_cycle.cycle_key`。
- `cycle_ganzhi`: 必须等于 locked facts 中的 `selected_cycle.ganzhi`。
- `title`
- `score_tendency`
- `core_theme`
- `opportunities`
- `risks`
- `action_guidance`
- `elements_check`: 必须包含 `原局`、`大运`、`十神`、`五行`、`合冲刑害`、`喜忌`。
