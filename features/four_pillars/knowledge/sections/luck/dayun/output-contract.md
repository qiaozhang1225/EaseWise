# Dayun Output Contract

必须输出 JSON object：
- `cycle_key`: 必须等于 locked facts 中的 `selected_cycle.cycle_key`。
- `cycle_ganzhi`: 必须等于 locked facts 中的 `selected_cycle.ganzhi`。
- `title`
- `trend_tendency`
- `core_theme`
- `opportunities`
- `risks`
- `action_guidance`
- `elements_check`: 必须包含 `原局`、`大运`、`十神`、`五行`、`合冲刑害`、`喜忌`。

主文案字段写法：
- `core_theme` 只写这十年的主轴判断。
- `opportunities` 只写机会、资源、助力和可推进的现实方向。
- `risks` 只写压力、消耗、冲突和需要避开的现实问题。
- `action_guidance` 只写具体行动建议。
- 上述字段都写成自然段，不要使用 `1、2、3`、`①②③`、项目符号或小标题列表。
- 每个字段 1 段，最多 2 句；不要在绿色机会段和红色风险段里重复同一套编号结构。
