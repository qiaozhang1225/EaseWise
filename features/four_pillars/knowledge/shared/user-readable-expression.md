# Four Pillars User-Readable Expression

## Goal

四柱八字的用户可见文案要让用户读完能马上对照自己的现实状态，而不是只觉得“很玄很专业”。

专业词可以保留，但它必须服务于理解。用户看到 `伤官`、`七杀`、`正印`、`夫妻宫`、`空亡`、`墓库`、`格局` 等词时，应当立刻知道它大概对应什么行为、关系、压力或资源。

## Public Paragraph Rules

`title`、`content`、`risk`、`usage_guidance`、大运/流年主段落必须：

- 先说现实判断，再补命理依据。
- 每段最多自然带出 1-2 个核心术语，避免连续堆叠。
- 术语第一次出现时必须紧跟白话解释。
- 解释必须落到性格、关系、工作、钱、健康、家庭、合作、收尾、情绪或行动节奏。
- 不要用术语开头或结尾。
- 不要把 `elements_check` 逐项抄进主段落。
- 不要把“结构承接、可用空间、现实承接感、底层结构、后段承接”这类内部分析词写进用户主段落。

推荐句式：

- `这里有“伤官”信号，也就是表达直接、主见强，现实里容易提出不同意见，也容易和规则或权威顶上。`
- `夫妻宫受冲，意思是亲密关系的相处节奏容易被外部变化打断，现实里要注意沟通、距离和家庭事务的牵动。`

## Elements Check Rule

`elements_check` 可以保留专业层，但每项也必须写成“术语判断 + 白话解释”，不要只写标签。

- 好：`十神里伤官较明显，伤官代表表达和突破，现实里容易说得直接、想按自己的方法做事。`
- 差：`伤官明显，结构有冲。`
