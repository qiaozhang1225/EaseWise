# Four Pillars Professional Terms Protocol

## Core Rule

专业词要保留，但不能裸露成结论。四柱输出必须遵守：

- 先点名术语，再马上翻译成白话，再落到现实场景。
- 不能只写“某某格局好/差”“伤官明显”“空亡较重”。
- 用户主文案每段最多自然带出 1-2 个核心术语。
- 如果术语解释后仍然很硬，就只写现实表现，把术语放到 `elements_check`。

## Must-Explain Four Pillars Terms

- `伤官`：表达强、主见重、反应快，容易挑战规则或说话太直接。
- `食神`：表达较柔和、会输出、会照顾体验，也代表技能和持续产出。
- `正印`：学习、资质、保护、长辈或平台支持。
- `偏印`：独立思考、钻研、敏感和非标准路径，也可能想得多。
- `七杀`：压力、竞争、执行力，也可能代表高压环境或强约束。
- `正官`：规则、职位、责任和稳定秩序。
- `官杀`：规则与压力系统，现实里对应上级、制度、责任、竞争。
- `财星`：钱、资源、现实交换、伴侣资源和经营能力。
- `比劫`：同辈、竞争、自我主张、兄弟姐妹和资源分配。
- `夫妻宫`：亲密关系里的相处位置，现实里看沟通、边界和共同生活节奏。
- `墓库`：资源被收住或压住，也可能在被触发时释放成果、钱或责任。
- `禄神`：职位、收入、贵人和稳定资源，不等于必然发财。
- `空亡`：事情容易落空、拖延、兑现感弱，或需要反复确认。
- `格局`：命盘里主要力量如何配合，现实里对应资源获取、表达方式、压力处理和成长路径。
- `喜忌`：更适合调动或节制的方向，不能写成绝对吉凶。
- `合冲刑害`：变化、拉扯、摩擦和关系牵动，不要写成确定灾祸。

## Internal Phrases Not For Public Copy

下面这些是内部分析词，不要出现在用户主文案中：

- 结构承接
- 可用空间
- 可用之处
- 现实承接感
- 底层结构
- 后段承接
- 专业层
- 输出合同
- locked facts

可替代表达：

- `结构承接` -> `现实里能不能稳住机会、把事情做成`
- `可用空间` -> `有优势，但要注意怎么使用`
- `后段承接差` -> `开始不难，但收尾、关系维护或结果兑现容易反复`
- `格局高` -> `这组力量配合得比较顺，现实里更容易把能力用出来`
