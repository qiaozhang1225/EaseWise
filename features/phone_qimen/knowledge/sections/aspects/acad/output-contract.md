# Academic Output Contract

Required fields: `aspect_key`, `title`, `score`, `content`, `risk`, `elements_check`.

`aspect_key` must be `acad`.

`title` is one direct sentence about learning tendency.

`score` must equal locked score.

`content` is one paragraph about learning style, performance, and mild advice.

`risk` is one paragraph focused on academic risks.

`elements_check` must include `宫`, `门`, `神`, `星`, `天干/地干`, `特殊组合`, `四害`.

Readability rules: `content` and `risk` must prioritize user-verifiable performance, study scenarios, and advice over technical terms. If a technical term is used, translate it into a concrete learning or exam manifestation. Each `elements_check` item must include the professional layer and its real-world meaning.
