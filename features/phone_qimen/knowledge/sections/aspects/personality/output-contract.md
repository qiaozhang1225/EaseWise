# Personality Output Contract

Required fields: `aspect_key`, `title`, `score`, `content`, `risk`, `elements_check`.

`aspect_key` must be `personality`.

`title` is one direct sentence about personality.

`score` must equal locked score.

`content` is one paragraph describing verifiable personality traits and mild advice.

`risk` is one paragraph focused on personality risks.

`elements_check` must include `宫`, `门`, `神`, `星`, `天干/地干`, `特殊组合`, `四害`.

Readability rules: `content` and `risk` must prioritize user-verifiable habits, reactions, communication style, and decision patterns over technical terms. If a technical term is used, translate it into a concrete personality manifestation. Each `elements_check` item must include the professional layer and its real-world meaning.
