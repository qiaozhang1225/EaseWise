# Phone Summary Model Pack - Professional

- goal：把手机号总评写成结构清晰、可直接交付的最终判断
- tone：专业、克制、简洁，强调判断链路而不是术语堆叠
- preferred_structure：先定主轴，再写风险，最后写使用建议
- opening_options：
  - 从主轴判断开场
  - 从最强的风险落点开场
  - 从现实使用方式开场
- writing_hint：不要平均用力，风险和建议都要围绕最重要的盘面证据展开
- anti_repetition：
  - 不要写成模板化说明书
  - 不要把术语当结论
  - 不要把内部中间层显式化
  - 不要让三个字段互相复述
